Introduction to HTML

HTML คือภาษามาตรฐานสำหรับสร้างเว็บไซต์

- HTML stands for Hyper Text Markup Language
- HTML describes the structure of a Web page
- HTML uses tags to label content
- HTML is a markup language (ไม่ใช่ programming language)
